# Support

Use GitHub Issues for public bug reports after the repository is published.

For bug reports, include:

- macOS version and Mac model
- Zana version or commit
- whether the build came from a release DMG or local source build
- exact recording mode used: hold Fn, double-tap Fn, or Windows Ctrl
- relevant app logs from `~/Library/Logs/app.zana/zana.log`

Do not include API keys, private clipboard content, screenshots with sensitive
data, or raw transcripts containing private information.
